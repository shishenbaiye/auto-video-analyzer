# Auto Video Analyzer Skill

自动视频分析技能 - 一键提取视频帧并进行AI视觉分析。

---

## 使用方法

当你想让我分析视频时，直接说：

```
分析视频 C:\Users\...\video.mp4
```

或

```
看一下这个视频：xxx.mp4，告诉我里面有什么
```

我会自动：
1. 复制视频到临时工作区
2. 智能提取关键帧
3. 分析每一帧内容
4. 给你完整报告
5. 自动清理临时文件

---

## 分析模式

| 模式 | 触发方式 | 用途 |
|------|---------|------|
| **Scan** (默认) | "分析视频" | 快速了解视频整体内容，提取~8帧 |
| **Full** | "详细分析视频" | 深度分析，开头密集+中间采样，~16帧 |
| **Debug** | "Debug视频...问题" | 连续帧分析，用于排查问题，每秒2帧 |

---

## 技术流程

```
用户请求
    ↓
检测视频路径
    ↓
复制视频到 temp/{id}/
    ↓
FFmpeg 提取帧
    ↓
AI 分析帧图片
    ↓
生成分析报告
    ↓
清理 temp/{id}/
    ↓
返回结果
```

---

## 依赖

- FFmpeg (自动调用已安装的版本)
- PowerShell
- 支持的视频格式：mp4, avi, mov, webm 等

---

## 示例

### 基础分析
```
你：分析视频 D:\recordings\gameplay.mp4
我：已提取8帧，这是三国SLG游戏，展示了战斗场景...
```

### Debug模式
```
你：Debug视频 D:\recordings\bug.mp4，为什么角色不移动
我：已提取连续帧，从第12帧开始角色位置没有变化，可能是动画状态机问题...
```

### 详细分析
```
你：详细分析视频 D:\recordings\demo.mp4
我：已提取16帧，完整分析UI布局、交互流程、视觉反馈...
```

---

## 注意事项

- 视频文件会被临时复制到 workspace/temp/ 目录
- 分析完成后自动清理，不会占用磁盘空间
- 大视频（>100MB）可能需要更长时间
- 如果视频路径包含空格，不需要加引号
